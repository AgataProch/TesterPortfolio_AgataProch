# Web Testing Project

This folder contains a simple **HTML & CSS website** created and tested as part of my QA training at **Czechitas Digital Academy: Testing**.  
The project demonstrates my understanding of frontend basics, structure validation, and visual inspection techniques.

---

## Project Overview

**Goal:**  
Create a small, static webpage and verify its correct display, structure, and responsiveness using testing tools and techniques.

**Tech stack:**
- HTML (semantic structure, accessibility)
- CSS (styling, layout, responsiveness)
- Chrome DevTools (UI inspection, breakpoints, console)
- Visual Studio Code (development environment)

---
